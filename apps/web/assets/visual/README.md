# Player HUD 视觉资产约束

本轮只接入一套 `极简科幻竞技风` 资产，不允许多套候选并存。

## 视觉语言

- 深色竞技场背景，冷色主基调
- 蓝红阵营对比明确，禁止第三主色抢戏
- 线条干净，发光克制，避免廉价赛博描边堆砌
- UI 框体偏薄、圆角稳定、层级靠阴影和半透明区分
- 人物暂不做全量战斗单位美术，只做地图氛围、框体、徽记、目标点底板

## 生图强约束提示词模板

### 地图氛围图

`minimal sci-fi competitive arena background, top-down moba broadcast feel, dark navy battlefield, teal vs coral team energy, clean geometry, restrained glow, premium esports UI atmosphere, no characters, no text, no logos, no clutter, high consistency, sharp edges, elegant and minimal`

### UI 框体 / 目标点底板

`premium minimal sci-fi HUD frame, dark navy glass panel, thin neon edge, teal and coral accent only, geometric objective plate, no fantasy ornament, no grunge, no text, no screenshots, unified esports control-room aesthetic`

### 指挥官徽记 / 头像底图

`minimal commander crest icon, sci-fi tactical emblem, geometric shield, teal or coral energy accent, clean silhouette, premium broadcast package style, no face, no text, no clutter, vector-like clarity`

## 一致性检查清单

- 配色只允许 `深蓝 + 青绿 + 珊瑚红 + 琥珀点缀`
- 发光面积不能大于主体面积的 20%
- 阴影不能糊成大面积灰雾
- 不出现写实人物、枪械、城市景观、二次元脸
- 不出现多套不同透视和不同材质语言
- 新资产如果比当前程序化元素更花、更亮、更乱，判定为不通过
